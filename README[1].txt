12-WEEK STRENGTH + MOBILITY — iPHONE WEB APP

HOW TO USE ON AN IPHONE
1. Put this folder on any HTTPS web host (GitHub Pages, Netlify, your organization website, etc.).
2. Open the web address in Safari on your iPhone.
3. Tap Share.
4. Tap “Add to Home Screen.”
5. Open the new app icon.

The app works offline after the first visit. Workout completion, notes, week selection, and daily habit entries are stored only in the browser on that device.

For a quick desktop preview, open index.html. Some offline-install features require an HTTPS web address.
